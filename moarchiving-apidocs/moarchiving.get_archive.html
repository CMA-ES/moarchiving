moarchiving.get_archive

module documentation

This module contains the factory functions for creating MOArchive objects of the appropriate dimensionality, for both constrained and unconstrained problems.

Function get_cmo_archive Function for creating CMOArchive objects, with similar interface as get_mo_archive.

def get_cmo_archive(list_of_f_vals=None, list_of_g_vals=None, reference_point=None, infos=None, n_obj=None, tau=1): ¶

Function for creating CMOArchive objects, with similar interface as get_mo_archive. Args: list_of_f_vals: list of objective vectors, can be None if n_obj is provided list_of_g_vals: list of constraint vectors, must be the same length as list_of_f_vals reference_point: reference point for the archive infos: list of additional information for each objective vector n_obj: must be provided if list_of_f_vals is None tau: threshold that indicates when the indicator reaches feasibility Returns: MOArchive object of the appropriate dimensionality, based on the number of objectives
